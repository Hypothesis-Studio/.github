# Hypothesis Studio

> Building tools that make developers more productive.

---

## About

**Hypothesis Studio** is an open-source organization focused on building modern developer tools. Our goal is to create fast, extensible, and thoughtfully designed software that developers actually enjoy using.

## 🚀 Projects

| Project | Description |
|---------|-------------|
| 📝 **[Hypothesis Editor](https://github.com/Hypothesis-Studio/Hypothesis-Editor)** | Create and edit Hypothesis projects with an intuitive interface. |
| 🧩 **[Hypothesis Extension](https://github.com/Hypothesis-Studio/Hypothesis-Extension)** | Official collection of extensions for Hypothesis Editor. |
